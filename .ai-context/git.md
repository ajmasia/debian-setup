# Git

## Semantic commits

Allowed prefixes: `feat`, `fix`, `chore`, `refactor`, `docs`.

Format: `type: short description` — English, lowercase, no trailing period.

```
feat: add Obsidian module to Software > Productivity
fix: install gpg if missing before gum setup
chore: bump version to 0.13.8
refactor: move Utilities from Software to Shell Tools
docs: update README and CHANGELOG for v0.13.8
```

## Strict rules

- **NEVER** reference Claude AI in commit messages — no Co-Authored-By, no "generated by", no "assisted by", nothing
- **NEVER** use `--no-verify` or skip hooks
- **NEVER** force push to `main`

## Signing

Commits signed with SSH signing (ED25519), configured in the repo/user git config.

## Branches

- `develop` — daily work branch, all commits go here
- `main` — only receives merges from develop (releases)
- No intermediate feature branches
